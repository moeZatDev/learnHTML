Learning the concept and the purpose of HTML

What is HTML?

HTML (Hypertext Markup Language) is the standard markup language for creating web pages. HTML is not a programming language. HTML describes the structure and layout of a web page and HTML elements tell the browser how to display the content.

!DOCTYPE html

DOCTYPE stands for "Document Type Declaration". In HTML, a DOCTYPE declaration is used to specify which version of HTML is being used in the document. The DOCTYPE declaration should be placed at the very beginning of an HTML document, before the html tag. It is not technically an HTML tag, but rather an instruction to the web browser on how to interpret the document.

html tag

The html element represents the root (top-level element) of an HTML document, so it is also referred to as the root element. All other elements must be descendants of this element.

Document metadata

Metadata contains information about the page. This includes information about styles, scripts and data to help software (search engines, browsers, etc.) use and render the page.


head tag

The head HTML element contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets.


meta tag

The meta HTML element represents Metadata that cannot be represented by other HTML meta-related elements, like base, link, script, style or title.


link tag

The link HTML element specifies relationships between the current document and an external resource.


base tag

The base HTML element specifies the base URL to use for all relative URLs in a document. There can be only one element in a document.


title tag

The title HTML element defines the document's title that is shown in a Browser's title bar or a page's tab.


style tag

The style element contains style information for a document. It contains CSS, which is applied to the contents of the document.


body tag

The body HTML element represents the content of an HTML document.

Scripting Tags

script tag

The script element is used to embed executable code or data; this is typically used to embed or refer to JavaScript code.

noscript tag

The noscript element defines a section of HTML to be inserted if a script type on the page is unsupported or if scripting is currently turned off in the browser.

Bidirectional Isolate Element (bdi)

BDI (Bi-Directional Isolation) used to isolate text that might be formatted in a different direction than the surrounding text.
BDI tag is used to ensure that text written in a right-to-left script, such as Arabic or Hebrew, is displayed correctly within a left-to-right context, such as an English language webpage.
The BDI tag creates a "directionality isolation" that prevents the text from inheriting the directionality of the surrounding text. This means that the text within the BDI tag will be displayed in the correct direction, regardless of the direction of the surrounding text.

The (Date) Time Element (time)

The time HTML element represents a specific period in time. It may include the datetime attribute to translate dates into machine-readable format, allowing for better search engine results or custom features such as reminders.

The Data Element (data)

The data tag is used to add a machine-readable translation of a given content.
This element provides both a machine-readable value for data processors, and a human-readable value for rendering in a browser.

The Ruby Annotation Element (ruby / rt / rp)

The ruby HTML element represents small annotations that are rendered above, below, or next to base text, usually used for showing the pronunciation of East Asian characters.
It can also be used for annotating other kinds of text, but this usage is less common.
